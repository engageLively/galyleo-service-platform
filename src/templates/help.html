Using the Global Data Plane Wiki

The Global Data Plane Wiki is primarily designed for use with API access, not browser access. This UI is designed for debugging/explanatory access.

To access hosted data, use one of the supported routes below with any standard HTTP request library; e.g. requests in Python, fetch in JavaScript, or with an executable such as wget or cURL. The result is always in a JSON list or (in the case of get_filtered_rows) as a list of lists.

The full documentation on the protocol is in SDTP Documentation

UC Berkeley users can upload new tables to the Wiki. The tables must be in Simple Data Markup Language format. At the moment, RowTables and RemoteTables are supported.
